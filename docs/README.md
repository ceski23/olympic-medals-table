> **Note**

## Development

1. Clone project
2. Install dependencies: `npm install`
3. Run project: `npm run dev`
4. In browser open: [http://localhost:5173](http://localhost:5173)

## Building app

1. Run: `npm run build`
2. App will be build in `dist` directory

## Testing

1. To start tests run: `npm run test`
